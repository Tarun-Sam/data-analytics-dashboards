## Project Overview
This project presents an interactive Excel dashboard analysing the performance of the
Argentina national football team during the FIFA World Cup 2022. The dashboard focuses
on individual player contributions and overall team performance across the tournament.

## Dataset
The dataset contains detailed player-level statistics for the Argentina squad in the
FIFA World Cup 2022, including appearances, goals, assists, dribbles, defensive actions,
and dual metrics.

## Dashboard Features
- Player-wise performance breakdown for the Argentina national team
- Goals and assists contribution analysis
- Defensive metrics such as tackles and interceptions per 90 minutes
- Interactive filters by position, player, and match participation
